PROGRAM SPECIFICATIONS

Create a Jupyter Notebook named Proj05YourName.ipynb.

The first section of the notebook must display the following certification:

I certify that this project is my own work and is not the work of others. I agree not to share my solution with others. - Print your name here.

Following that, the notebook must use SciPy to solve for x, y, and z for the following system of equations.

3x + 4y + 3z = 10
9x - 10y - 10z = -11
4x + 15y +18z = 37

The notebook must include Markdown sections that explain each step in the solution of the problem. The code sections must print all intermediate results such as the values of matrices, etc.

The notebook must include a code section that confirms that the solution is correct along with a Markdown section explaining how the code confirms that the solution is correct.

The final section must be a Markdown section that shows the solution in the following format

x = ??

y = ??

z = ??

where the actual solution values replace the ?? shown above.

Download the notebook into an HTML file. Encapsulate a copy of your output HTML file and a copy of the ipynb file in a zip file and submit the zip file via Blackboard.

Be sure that your name appears in the certification as indicated.
